# Home

Welcome to the CasaYes Documentation Hub!

This guide will walk you through the essentials of integrating with the CasaYes platform via our Integration methods and Webhooks. Here’s what you need to know:

## Overview

* **Webhooks**: Stay up-to-date with real-time notifications. Webhooks alert your application about specific events as they occur, ensuring your system always has the latest information. For further details, refer to our [Webhooks Documentation](webhooks/webhooks.md).
* **Integration**: Our integration methods enable seamless data transfer to the CasaYes platform. You can use APIs for real-time, structured data exchange or FTP for large file transfers and batch processing. Both methods ensure that your information is efficiently communicated to our system. For more details, refer to our [Integration Documentation](integration/integration-documentation.md).
