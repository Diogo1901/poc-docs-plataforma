# Início

Bem-vindo ao Centro de Documentação da CasaYes!

Este guia irá acompanhá-lo através dos princípios essenciais para integrar com a plataforma CasaYes, utilizando os nossos métodos de Integração e Webhooks. Eis o que precisa de saber:

## Visão Geral

* **Webhooks**: Mantenha-se atualizado com notificações em tempo real. Os webhooks alertam a sua aplicação sobre eventos específicos à medida que ocorrem, garantindo que o seu sistema dispõe sempre da informação mais recente. Para mais detalhes, consulte a nossa [Documentação de Webhooks](webhooks/webhooks.md).
* **Integração**: Os nossos métodos de integração permitem uma transferência de dados simples e eficiente para a plataforma CasaYes. Pode utilizar APIs para uma troca de dados estruturada e em tempo real, ou FTP para transferências de ficheiros de grande dimensão e processamento em lote. Ambos os métodos asseguram que a sua informação é comunicada de forma eficaz ao nosso sistema. Para mais detalhes, consulte a nossa [Documentação de Integração](integracao/documentacao-de-integracao.md).
