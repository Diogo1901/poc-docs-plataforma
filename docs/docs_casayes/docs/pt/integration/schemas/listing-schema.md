# Imóvel

## Modelo de Imóvel

| Nome             | Tipo       | Descrição                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| ---------------- | ---------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `type`           | `string`   | <p>Tipo de imóvel. Um de <code>[Apartment, House, Building, Farm, Land, BusinessOffice, Store, Warehouse, Garage, HouseFloor, SaleOfBusiness]</code><br><code>required</code></p>                                                                                                                                                                                                                                                               |
| `subType`        | `string`   | Subtipo do imóvel. Um de `[UrbanLand, BuildableLand, RusticLand, IndustrialLand]`                                                                                                                                                                                                                                                                                                                                                               |
| `reference`      | `string`   | <p>Referência interna do imóvel no seu CRM. Deve ser única.<br><code>required</code><br><code>minLength: 1</code><br><code>string ^.{0,100}$</code></p>                                                                                                                                                                                                                                                                                         |
| `referenceURL`   | `string`   | <p>Link público para o imóvel.<br><code>nullable: true</code><br><code>string ^.{0,150}$</code></p>                                                                                                                                                                                                                                                                                                                                             |
| `address`        | `object`   | <p><a href="#modelo-de-morada">Modelo de Morada</a><br>Morada do imóvel; é obrigatório enviar as coordenadas do imóvel ou o código postal, mesmo que decida ocultar a localização exata no site.<br><code>required</code></p>                                                                                                                                                                                                          |
| `contact`        | `object`   | <p><a href="#modelo-de-contacto">Modelo de Contacto Modelo</a><br>Informações de contacto para o imóvel. Por defeito, as informações de contacto serão retiradas da Agência a que o imóvel pertence. Opcionalmente, é possível usar este objeto para substituir estas informações e atribuir automaticamente o imóvel a um utilizador fornecendo o email do utilizador registado no Portal CasaYes.<br><code>nullable: true</code></p> |
| `attributes`     | `object`   | <p><a href="#modelo-de-atributos">Modelo de Atributos</a><br>Informações principais do imóvel.<br><code>required</code></p>                                                                                                                                                                                                                                                                                                            |
| `operation`      | `object`   | <p><a href="#modelo-da-operacao">Modelo da Operação</a><br>Informações relacionadas com o preço e comissões do imóvel.<br><code>required</code></p>                                                                                                                                                                                                                                                                                    |
| `descriptions`   | `object[]` | <p><a href="#modelo-das-descricoes">Modelo das Descrições</a><br>Lista de descrições a mostrar no detalhe do imóvel.<br><code>required</code></p>                                                                                                                                                                                                                                                                                      |
| `rooms`          | `object[]` | <p><a href="#modelo-das-divisoes">Modelo das Divisões</a><br>Lista de divisões a mostrar no detalhe do imóvel.<br><code>nullable: true</code></p>                                                                                                                                                                                                                                                                                      |
| `images`         | `object[]` | <p><a href="#modelo-das-imagens">Modelo das Imagens</a><br>Lista de imagens a mostrar na página de detalhe do imóvel. A ordem das imagens será automaticamente definida pela ordem das imagens nesta lista. Opcionalmente, o campo "order" pode ser preenchido para especificar uma ordem personalizada. A imagem principal do imóvel será sempre a primeira imagem, em qualquer caso.<br><code>nullable: true</code></p>              |
| `videos`         | `object[]` | <p><a href="#modelos-dos-videos">Modelo dos Vídeos</a><br>Lista de vídeos a mostrar no detalhe do imóvel.<br><code>nullable: true</code></p>                                                                                                                                                                                                                                                                                           |
| `virtualTourUrl` | `string`   | <p>Tour virtual a mostrar no detalhe do imóvel.<br><code>nullable: true</code><br><code>string ^.{0,500}$</code></p>                                                                                                                                                                                                                                                                                                                            |
| `buildingName`   | `string`   | <p>Nome do empreendimento ao qual o imóvel pode pertencer.<br><code>nullable: true</code><br><code>string ^.{0,100}$</code></p>                                                                                                                                                                                                                                                                                                                 |
| `privateNotes`   | `string`   | <p>Notas privadas do imóvel. Estas notas privadas podem ser acedidas ao visitar o detalhe do imóvel no Portal CasaYes.<br><code>nullable: true</code><br><code>string ^.{0,1000}$</code></p>                                                                                                                                                                                                                                                    |
| `overrideTitle`  | `string`   | <p>Título alternativo a mostrar no site público antes da descrição.<br><code>nullable: true</code><br><code>string ^.{0,100}$</code></p>                                                                                                                                                                                                                                                                                                        |

#### Modelo de Morada

| Nome          | Tipo               | Descrição                                                                                                                                                                                          |
| ------------- | ------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `visibility`  | `boolean`          | Se a morada do imóvel é público. Por defeito, verdadeiro.                                                                                                                                          |
| `street`      | `string`           | <p>Nome da rua do imóvel.<br><code>nullable: true</code></p>                                                                                                                                       |
| `floor`       | `string`           | <p>Andar do imóvel.<br><code>nullable: true</code><br><code>pattern: -[1-2]|[1-9]|[1-5][0-9]|60|rc|cv|scv|ult</code><br>Onde: rc = Rés-do-chão, cv = Cave, scv = Semi-cave, ult = Último piso.</p> |
| `town`        | `string`           | <p>Nome da cidade do imóvel.<br><code>nullable: true</code></p>                                                                                                                                    |
| `doorNumber`  | `integer ($int32)` | <p>Número da porta do imóvel.<br><code>nullable: true</code></p>                                                                                                                                   |
| `zipCode`     | `string`           | <p>Código postal do imóvel. Obrigatório se não fornecer as coordenadas do imóvel.<br><code>nullable: true</code><br><code>string ^.{0,10}$</code></p>                                              |
| `country`     | `string`           | País do imóvel. Um de `[Portuguese, Spanish, French, English, German, Italian]`                                                                                                                    |
| `coordinates` | `object`           | <p>Modelo de Coordenadas de Morada<br>Coordenadas do imóvel. Obrigatório se não enviar o código postal.<br><code>nullable: true</code></p>                                                         |

**Modelo de Coordenadas de Morada**

| Nome        | Tipo               | Descrição                                                                                                                      |
| ----------- | ------------------ | ------------------------------------------------------------------------------------------------------------------------------ |
| `latitude`  | `number ($double)` | <p>Latitude do imóvel. Precisão melhora com mais decimais.<br><code>required</code><br><code>number [-90 ... 90]</code></p>    |
| `longitude` | `number ($double)` | <p>Longitude do imóvel. Precisão melhora com mais decimais.<br><code>required</code><br><code>number [-180 ... 180]</code></p> |

#### Modelo de Contacto

| Nome          | Tipo               | Descrição                                                                                                                                                                                                                                     |
| ------------- | ------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `name`        | `string`           | <p>Nome para substituir as informações da agência no site.<br><code>nullable: true</code></p>                                                                                                                                                 |
| `email`       | `string ($email)`  | <p>Email de um utilizador no Portal CasaYes. Este email permite atribuir o imóvel ao utilizador. Se o utilizador não existir, este email irá substituir o email da agência no site.<br><code>required</code><br><code>minLength: 1</code></p> |
| `phonePrefix` | `integer ($int32)` | <p>Prefixo do país para substituir as informações da agência no site.<br><code>nullable: true</code><br><code>pattern: ^[1-9][0-9]{1,3}$</code></p>                                                                                           |
| `phone`       | `integer ($int64)` | <p>Telefone para substituir as informações da agência no site.<br><code>nullable: true</code><br><code>pattern: ^[1-9][0-9]{4,15}$</code></p>                                                                                                 |

#### Modelo de Atributos

| Nome                           | Tipo                  | Descrição                                                                                                                                                                                                                                                             |
| ------------------------------ | --------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `plotArea`                     | `integer ($int32)`    | <p>Área do lote.<br><code>nullable: true</code></p>                                                                                                                                                                                                                   |
| `builtArea`                    | `integer ($int32)`    | <p>Área bruta.<br><code>nullable: true</code></p>                                                                                                                                                                                                                     |
| `totalArea`                    | `integer ($int32)`    | <p>Área bruta privativa.<br><code>required</code></p>                                                                                                                                                                                                                 |
| `usableArea`                   | `integer ($int32)`    | <p>Área útil.<br><code>nullable: true</code></p>                                                                                                                                                                                                                      |
| `maxConstructionArea`          | `integer ($int32)`    | <p>Área máxima para construção.<br><code>nullable: true</code></p>                                                                                                                                                                                                    |
| `floors`                       | `integer ($int32)`    | <p>Número de pisos.<br><code>nullable: true</code></p>                                                                                                                                                                                                                |
| `characteristics`              | `string[]`            | <p>Lista com características do imóvel.<br>Lista de <code>[ AirConditioning, EquippedKitchen, Balcony, Terrace, StorageRoom, Garden, SwimmingPool, Heating, ElectricCarsCharging, BoxGarage, OutdoorParking, SharedGarage ]</code><br><code>nullable: true</code></p> |
| `depositMonths`                | `integer ($int32)`    | <p>Apenas para arrendamento. Meses de caução exigidos.<br><code>nullable: true</code></p>                                                                                                                                                                             |
| `parking`                      | `boolean`             | Se existe estacionamento disponível. "Parking" refere-se a "Garagem"; para outros tipos de estacionamento use `characteristics`.                                                                                                                                      |
| `parkingCapacity`              | `integer ($int32)`    | <p>Capacidade do estacionamento (garagem). Apenas se <code>parking</code> for verdadeiro.<br><code>nullable: true</code></p>                                                                                                                                          |
| `builtYear`                    | `integer ($int32)`    | <p>Ano de construção do imóvel.<br><code>nullable: true</code><br><code>number [999 ... 2100]</code></p>                                                                                                                                                              |
| `conservation`                 | `string`              | <p>Estado de conservação do imóvel.<br>Um de <code>[ New, Renovated, GoodGeneralConditions, RefreshmentsToPredict, WorkNeeded, WorkInProgress ]</code><br><code>nullable: true</code></p>                                                                             |
| `energyCertificateRating`      | `string`              | <p>Classe energética do imóvel.<br>Um de <code>[ APlus, A, B, BMinus, C, D, E, F, NA ]</code><br><code>required</code></p>                                                                                                                                            |
| `energyCertificatePerformance` | `number ($double)`    | <p>Desempenho energético do imóvel.<br><code>nullable: true</code></p>                                                                                                                                                                                                |
| `bathrooms`                    | `integer ($int32)`    | <p>Número de casas de banho.<br><code>nullable: true</code></p>                                                                                                                                                                                                       |
| `bedrooms`                     | `integer ($int32)`    | <p>Número de quartos.<br><code>nullable: true</code></p>                                                                                                                                                                                                              |
| `rooms`                        | `integer ($int32)`    | <p>Número de divisões.<br><code>nullable: true</code></p>                                                                                                                                                                                                             |
| `parkingIncludedInPrice`       | `boolean`             | Se o estacionamento (garagem) está incluído no preço do imóvel. Apenas disponível se `parking` for verdadeiro.                                                                                                                                                        |
| `parkingPrice`                 | `number ($decimal)`   | <p>Valor do estacionamento (garagem). Apenas disponível se <code>parking</code> for verdadeiro e <code>parkingIncludedInPrice</code> for falso.<br><code>nullable: true</code></p>                                                                                    |
| `minimalStay`                  | `integer ($int32)`    | <p>Apenas para arrendamento. Meses mínimos de permanência após o arrendamento.<br><code>nullable: true</code></p>                                                                                                                                                     |
| `availableFrom`                | `string ($date-time)` | <p>Apenas para arrendamento. Data em que o imóvel ficará disponível.<br><code>nullable: true</code></p>                                                                                                                                                               |
| `minAge`                       | `integer ($int32)`    | <p>Apenas para arrendamento. Idade mínima para arrendar.<br><code>nullable: true</code></p>                                                                                                                                                                           |
| `maxAge`                       | `integer ($int32)`    | <p>Apenas para arrendamento. Idade máxima para arrendar.<br><code>nullable: true</code></p>                                                                                                                                                                           |
| `bankProperty`                 | `boolean`             | Se o imóvel pertence a um banco. Por defeito, falso.                                                                                                                                                                                                                  |
| `elevator`                     | `boolean`             | Se o imóvel tem elevador. Por defeito, falso.                                                                                                                                                                                                                         |
| `landRegister`                 | `string`              | <p>Código do registo predial.<br><code>nullable: true</code><br><code>string ^.{0,100}$</code></p>                                                                                                                                                                    |
| `treasuryLandRegister`         | `string`              | <p>Código do registo predial da tesouraria.<br><code>nullable: true</code><br><code>string ^.{0,50}$</code></p>                                                                                                                                                       |
| `houseLicensing`               | `string`              | <p>Apenas para arrendamento.<br><code>nullable: true</code><br><code>string ^.{0,150}$</code></p>                                                                                                                                                                     |
| `condominiumValue`             | `number ($decimal)`   | <p>Valor do condomínio.<br><code>nullable: true</code></p>                                                                                                                                                                                                            |
